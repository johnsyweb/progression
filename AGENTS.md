# Agent skills

This repo is configured for Matt Pocock's agent skills framework. Agent skills extend the coding agent with specialized workflows for architecture, refactoring, QA, and issue triage.

## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues. See [docs/agents/issue-tracker.md](docs/agents/issue-tracker.md).

### Triage labels

Five canonical labels manage issue workflow: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See [docs/agents/triage-labels.md](docs/agents/triage-labels.md).

### Domain docs

Single-context layout with `CONTEXT.md` and `docs/adr/` at the repo root. See [docs/agents/domain.md](docs/agents/domain.md).
